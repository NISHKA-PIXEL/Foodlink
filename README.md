# FoodLink

FoodLink is a premium Flutter app concept for real-time surplus food redistribution. It uses a dark eco-tech visual language with green accents and a startup-grade design system.

## Run

```bash
flutter pub get
flutter run
```

## Architecture

- Clean, feature-based folder structure.
- Riverpod state management.
- Reusable premium UI widgets (`GlassCard`, `PremiumButton`, `PremiumShell`).
- Placeholder Firebase service and repository-driven data flow.
